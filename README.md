# Description 

Goal & brief overview

This project is to recreate a `First Person Shooter` 3D game like `Wolfenstein 3D`. As a graphic design program, we will manipulate windows, colors, events, fill shapes , etc.  This is a project to apply some pratical cpplications of mathematics, notably Raycasting
